Fundamentos do FrontEnd

Quais são as principais tecnlogias para um desenvolvimento do FrontEnd?

HTML

HTML é a base de qualquer página da web. Ele serve para estruturar o conteúdo, dizendo ao navegador o que é título, parágrafo, imagem, link, etc.

CSS

CSS é a linguagem usada para estilizar páginas web — ou seja, deixar o HTML bonito e organizado visualmente.

JavaScript

JavaScript é a linguagem que adiciona interatividade e comportamento às páginas web.
